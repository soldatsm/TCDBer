# Description

This sctipt allows to:

1. Download TCDB database
2. Make blastp database from TCDB db
3. Do blastp with your protein sequences against TCDB db
4. Add protein existance evidence from UniProt to blast result


## Dependency
* `pandas`
* `argparse`
* `tqdm`
* `json`
* `rquests`
* `shutil`
* `sys`
* `subprocess`
